``<widget-model-disable-nbrecords-computation>`` is a backend widget that makes possible to disable the computation of nbRecords on domain fields.
See `source repository <https://github.com/oca/web>`_ 

**Table of contents**

.. contents::
   :local:

Usage
=====

Add ``widget="domain"`` to your domain field in form view. Optionally you can set ``autoFetch`` (boolean) attributes.
You can also set a res_company preference to change the default behaviour of the domain widget
