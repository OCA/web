==========================================
Web Digitized Signature for Stock Pickings
==========================================

This module adds a signature field (using ``web_widget_digitized_signature``)
to stock pickings report.

Also a button has been added to picking form to allow users to send by email
the picking report with digitized signature, if the picking is in state done.

Configuration
=============

#. Go to *Settings > Users > Users* and set a digital signature for a user.
#. Go to *Inventory > All Transfers* and print a picking order. Signature will
   be added to picking report.
