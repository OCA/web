================================
2D matrix for x2many fields demo
================================

This module prepares an example for the module 'web_widget_x2many_2d_matrix'.

It implements a simple wizards that allows users to enter the planned hours
for all the projects.

Usage
=====

Go to 'Project / Enter Planned hours'
