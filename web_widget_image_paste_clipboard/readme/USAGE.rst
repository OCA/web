To use this module, you need to:

#. Copy an image (from : Paint, Gimps, Google Image, ...)
#. Edit a record with image field
#. Click on the paste Icon (between Edit icon and Delete Icon)
#. Press Ctrl + V (or Cmd + V on MacOS)
