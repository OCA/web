========================
Web Drag Drop Attachment
========================

This module enables the feature to Drag & Drop the files in form view and create attachment.

Usage
=====

The attachments which are selected can simply drag & drop to the form view and
will be available at the "Attachments" dropdown box on the top of the form view.
