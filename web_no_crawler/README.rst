========================================
Implement robots.txt / disallow indexing
========================================

This module was written to implement a robots.txt file to prevent web crawlers (like google) from indexing pages.
This module does not depend on Website module.

More info about robots.txt: http://www.robotstxt.org/

Configuration
=============

No configuration needed.

Usage
=====

Once installed adds robots.txt (ex.: http://example.org/robots.txt).
